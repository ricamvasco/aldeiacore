# Análise Visual da Marca Aldeia

## Elementos Visuais Identificados
- **Árvore principal**: Ilustração line-art de árvore com raízes profundas, folhas, estrela no topo, coração e gráfico de crescimento
- **Estágios de crescimento**: 7 estágios (semente → broto → muda → planta → arbusto → árvore jovem → árvore frondosa)
- **Estilo**: Line-art delicado, traços finos, estilo botânico/orgânico
- **Ícones integrados**: coração (cuidado), estrela (excelência), gráfico crescente (evolução)

## Paleta de Cores
- Verde escuro (folhas/marca): #4A6741
- Marrom (tronco/terra): #8B7355 / #3b2f1e
- Off-white/creme (fundo): #FAF6F0
- Laranja/terra (acento): #E65100

## Tipografia
- DM Sans (principal)
- Estilo: clean, profissional, acolhedor

## Slogan
- "Porque nenhuma criança deveria crescer sozinha — é preciso uma aldeia inteira para florescer."
- "Conectar é desenvolver. Desenvolver é florescer."

## Conceito da Marca
- Crescimento orgânico (semente → árvore)
- Raízes profundas = base sólida de cuidado
- Comunidade/aldeia = suporte coletivo
- Foco: crianças com TEA e suas famílias
