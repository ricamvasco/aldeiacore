# Banner de Fachada — Clínica Aldeia
## Especificações Técnicas para Orçamento

---

## Proposta 1: Banner Horizontal (Lona)

Esta proposta é ideal para fachadas amplas ou muros laterais, com boa visibilidade para quem passa de carro ou a pé. O formato paisagem permite distribuir a informação de forma equilibrada, com a árvore da marca à esquerda, os serviços ao centro, e o QR Code à direita.

| Especificação | Detalhe |
|---|---|
| **Dimensão** | 3,00m (largura) × 1,50m (altura) |
| **Material sugerido** | Lona vinílica 440g com acabamento fosco |
| **Acabamento** | Ilhós a cada 30cm nas bordas + reforço nas pontas |
| **Resolução mínima** | 150 DPI (arquivo final: 17.717 × 8.858 px) |
| **Fixação** | Estrutura metálica ou fixação direta em muro/tapume |
| **Custo estimado** | R$ 180–350 (impressão) + R$ 200–500 (instalação) |

**Quando usar:** Fachada frontal do imóvel com espaço horizontal disponível, muro de divisa, ou tapume de obra. Excelente relação custo-benefício para alta visibilidade.

---

## Proposta 2: Placa Vertical (Totem/Cavalete)

Esta proposta é ideal para espaços mais estreitos, calçadas, ou como complemento à fachada. O formato retrato permite uma leitura natural de cima para baixo, com hierarquia clara: árvore → título → serviços → slogan → QR Code → contato.

| Especificação | Detalhe |
|---|---|
| **Dimensão** | 1,20m (largura) × 1,80m (altura) |
| **Material sugerido** | Placa de ACM (alumínio composto) ou PVC expandido 10mm |
| **Acabamento** | Impressão UV direta + laminação fosca anti-reflexo |
| **Resolução mínima** | 150 DPI (arquivo final: 7.087 × 10.630 px) |
| **Fixação** | Estrutura tipo cavalete/totem ou fixação em parede |
| **Custo estimado** | R$ 250–500 (placa ACM) ou R$ 150–300 (PVC) + instalação |

**Quando usar:** Calçada em frente ao imóvel (tipo cavalete), ao lado da porta de entrada, ou em espaços verticais estreitos. Mais durável que lona e com acabamento premium.

## Elementos Visuais Utilizados

A identidade visual segue rigorosamente a marca Aldeia, sem incluir o nome "Aldeia" de forma explícita (conforme orientação do projeto). Os elementos são:

| Elemento | Descrição |
|---|---|
| **Árvore** | Ilustração line-art com raízes profundas, folhas, estrela no topo e coração — símbolo da marca |
| **Paleta** | Fundo off-white (#FAF6F0), verde escuro (#4A6741), marrom quente (#8B7355) |
| **Tipografia** | Serifada para títulos (impacto), sans-serif para corpo (legibilidade) |
| **Slogan** | "Conectar é desenvolver. Desenvolver é florescer." (Proposta 1) |
| | "Porque nenhuma criança deveria crescer sozinha — é preciso uma aldeia inteira para florescer." (Proposta 2) |
| **QR Code** | Espaço reservado (branco com borda verde) — será gerado quando a landing page estiver pronta |
| **Contato** | WhatsApp (41) 99999-9999 + Instagram @clinica.aldeia |

## Recomendações para Produção

O QR Code deve ser impresso com no mínimo 15cm × 15cm na Proposta 1 e 12cm × 12cm na Proposta 2, garantindo leitura a até 2 metros de distância. Recomenda-se testar a leitura do QR Code impresso antes da instalação final.

Para a versão final de produção, será necessário substituir o placeholder do QR Code pelo código real (gerado a partir da URL da landing page) e confirmar o número de WhatsApp correto. O arquivo deve ser enviado à gráfica em formato PDF/X-1a com sangria de 2cm em todas as bordas.

A escolha entre lona (Proposta 1) e ACM/PVC (Proposta 2) depende da durabilidade desejada: lona é mais econômica mas dura 1-2 anos exposta ao sol; ACM dura 5+ anos com qualidade visual superior.
