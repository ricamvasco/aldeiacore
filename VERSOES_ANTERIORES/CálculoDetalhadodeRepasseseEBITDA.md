## 🎯 FÓRMULAS APLICADAS

### Repasse em GRUPOS (Psicóloga)
```
Repasse Grupos = % Nível Psicóloga x 30 atendimentos/semana x Ticket Médio x 0,30
Simplificado: % Nível x 30 x Ticket x 0,30
```

### Repasse em INDIVIDUAIS
```
Repasse = % Nível x Ticket Médio x Atendimentos Realizados
```

### Custos Variáveis
```
Custos Variáveis = Faturamento x 10%
```

### EBITDA
```
EBITDA = Faturamento - Total Repasses - Custos Fixos - Custos Variáveis
```

### Impostos
```
Impostos = EBITDA x 18%
```

### Lucro Líquido
```
Lucro Líquido = EBITDA - Impostos
```
