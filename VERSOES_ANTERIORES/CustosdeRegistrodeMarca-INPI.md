## 📅 PROCESSO DE REGISTRO

### Passo 1: Preparação (1 semana)
- Escolher tipo de marca (nominativa, figurativa ou mista)
- Definir descrição de produtos/serviços
- Preparar logo/design (se aplicável)
- **Custo:** R$ 0

### Passo 2: Protocolo (1 dia)
- Acessar site do INPI
- Preencher formulário de solicitação
- Fazer upload de documentos
- Pagar taxa
- **Custo:** R$ 575 (marca nominativa + 2 classes)

### Passo 3: Análise Preliminar (30-60 dias)
- INPI verifica se marca está disponível
- Verifica se há conflitos com marcas existentes
- **Custo:** R$ 0

### Passo 4: Publicação (30 dias)
- Se aprovado, marca é publicada no Diário da União
- Terceiros têm 60 dias para fazer oposição
- **Custo:** R$ 0

### Passo 5: Registro Final (30-60 dias)
- Se nenhuma oposição, marca é registrada
- Certificado é emitido
- **Custo:** R$ 0

**Tempo Total: 4-6 meses**
